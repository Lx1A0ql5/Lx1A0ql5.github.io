ip 地址是一段唯一标识，网络编码（二进制），32 位组成

# 32 位二进制形式的 IP 地址

在组建局域网等情况下，“连我 ip101010101101...”，已经死亡，为了便于记忆和操作，将八位 2 进制转化为十进制，成为点分十进制

# 点分十进制 IP 地址

X.X.X.X 的形式，其中 X 的取值范围是 0-255

# 子网掩码

局域网通信规则：在同一个局域网中 ，所有的 ip 必须在同一网段中才能通信

ip 地址构成：网络位 + 主机位（网络位相同的 IP 地址，为同一网段）

子网掩码如何确认网络位：与 255 对应的数字为网络位，与 0 对应的是主机位

注：主机位不能全为零，不能全为 255，主机位全部为 255 为网段的广播地址

例如

例如某 ip 11.10.11.1, 子网掩码为 255.255.0.0，则该 ip 的网络段为 11.10.0.0，主机用户名字为.11.1

# IP 地址详解

国际标准组织 ISO 定义地址分类：五大类（是以 IP 地址的第一位进行区分的）

A 类：1-126 默认子网掩码：255.0.0.0

（127.0.0.1 回环地址自己）

B 类：128-191 默认子网掩码：255.255.0.0

C 类：192-223 默认子网掩码：255.255.255.0

D 类：224-239 组播地址

E 类：240-254 科研使用

（C,D 不是普通人类用的，没有默认子网掩码）

我们能用的只有 ABC

（上述规定默认子网掩码已经推翻了，因为二三十年前科学家也没想到会有这么多电脑手机）

现在只作为一种习惯

# 网关
