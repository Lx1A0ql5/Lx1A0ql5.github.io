# 服务器 DOS 命令

netstat -an //查看本地所有开放的端口

组管理

在庞大的服务器中，给用户配置权限，只需要给它放入特定的组，例如 administrators，networks，users，guests

# 远程控制桌面

打开 win+r

在输入需要远程连接的 ip，前提是得获得可以远程控制权限的用户的账号密码。两边用户位于同一局域网中，且可以互相通信。

在需要被控制的电脑中，打开 cmd，输入 services.msc; 打开 Talnet 服务，属性设为自动。

在自己电脑中，cmd.

输入账号密码，即可在远程操控的电脑上进行命令操作。
